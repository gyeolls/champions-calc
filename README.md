# 포켓몬 챔피언스 데미지 계산기 (PC/웹)

예시 앱 「重ねて！ダメージ計算」(rakkolab) 분석을 바탕으로 PC용으로 재구현한 도구.
선출 화면에서 상대 6마리를 자동 인식하고, 실시간 채용률로 세트를 추정해
**데미지 · 피격 위험 · 행동 순서(스피드)** 를 한 번에 확인합니다.

## 실행

`index.html`을 **Chrome 또는 Edge**로 열면 동작 (더블클릭).
GitHub Pages로 호스팅하려면 `DEPLOY.md` 참조.

> OCR 최초 1회(한국어 모델)와 실시간 채용률 API·아이콘 CDN 폴백에 인터넷이 필요합니다.
> 오프라인이면 내장 데이터로 자동 폴백.

### 데이터 최신성
채용률은 championsbattledata.com의 **Current(최신) 데이터**를 시즌 지정 없이 호출 —
사이트가 새 일일 스냅샷을 올릴 때마다 자동으로 최신을 받습니다 (앱은 포켓몬별 6시간 캐시).

## 사용 흐름

### 1. 내 파티 등록 ("내 파티" 탭)
- 파티 여러 개 생성/이름변경/삭제 — 브라우저에 자동 저장되어 재방문 시 유지
- **이미지 등록**: 게임의 파티 → *능력* 화면과 *스테이터스* 화면을 캡처해 붙여넣기(Ctrl+V)/드롭
  - 능력 화면 → 특성·아이템·기술 4개 자동 등록
  - 스테이터스 화면 → 노력치 포인트(실수치에서 역산) + **성격(빨강↑/파랑↓ 화살표 색 인식)**
- 수동 수정: 계산기 탭에서 성격은 게임 트레이닝 화면과 같은 **5×5 그리드**로 선택

### 2. 상대 인식 ("계산기" 탭)
- 상단 바에 **선출/배틀 화면을 Ctrl+V** 붙여넣기 (또는 이미지 드래그)
  - 선출 화면 → 상대 6마리 스프라이트 매칭 (1차 색상 비교, 2차 **이로치**용 윤곽 상관 폴백)
  - 배틀 화면 → 이름 OCR
- 상대 팀 슬롯 클릭으로 대상 전환 (우클릭: 슬롯 비우기) — 기본 폼 우선
- **폼(메가 등)을 바꾸면 팀 슬롯에 고정**되어 스피드 비교표에도 그대로 반영
- 날씨/필드는 상대 패널 하단, 벽은 각 패널의 "벽 데미지 감소" 체크
- 화면 배치: 상대 팀 ↔ 내 포켓몬 사이 가운데에 양방향 데미지 표

### 3. 자동 세트 추정 (실시간 API)
- 상대 지정 시 **championsbattledata.com API**에서 최신 채용률(기술/아이템/성격/노력치/특성)을
  받아 1위 세트를 자동 적용 (6시간 캐시, 실패 시 내장 데이터)
- 조회는 항상 **기본 폼 우선** — 메가진화는 폼 드롭다운에서 수동 선택
- 리전폼·특성폼(알로라 나인테일, 돌핀맨 마이티 등)은 검색 목록에 중복 노출되지 않고
  대표 포켓몬의 "폼"에서 선택

### 4. 결과 읽기
- **스피드 비교**: 상대 팀 전원 × {최저/무보정/준속/최속/최속+스카프/추정 세트} vs 내 실수치.
  초록=선공, 빨강=후공, 노랑=동속. 순풍/트릭룸 토글 지원. 행 클릭 시 상대 지정
- **상대 → 나**: 채용률순 공격기 데미지 + 확정/난수 타수
- **나 → 상대**: 내 기술 4개 데미지 (선공기는 `선공+n` 표기)

### UI
- 아이템: 클릭 → 검색창 + 아이콘 리스트. **챔피언스 장착 가능 아이템(72종)만** 표시되고,
  메가스톤은 "메가스톤" 1개 항목으로 통합(아이콘은 해당 포켓몬 전용 스톤으로 표시)
- 기술: 클릭 → 해당 포켓몬 습득기만, 채용률 상위 → 타입별 → 가나다순.
  **위력·명중률**(필중 포함) 함께 표기 — 데미지 표에도 표시
- 신기술 한국어명은 최초 실행 시 PokeAPI에서 자동 보완(캐시)
- 내 파티 탭에서 슬롯/카드 클릭 → 그 자리에서 세부 편집 (파티 여러 개 저장)
- 헤더의 ☀️/🌙 버튼으로 라이트/다크 모드 전환 (자동 저장)

### 지원 해상도
- 선출 화면 인식: 720p~4K, 레터박스/필러박스/창모드(제목표시줄) 모두 검증 완료
- 파티 화면 등록: **1080p 이상 권장** (4K 기준 6/6, 1080p에서 글자가 흐린 일부
  포켓몬은 부분 인식될 수 있음 — 해당 슬롯만 클릭해서 수동 보완)

## 계산 지원 범위
Lv.50/IV31 고정, 챔피언스 포인트제(0~32). STAB·상성·랭크±6·급소·벽·화상·날씨 4종·
필드 4종(일렉트릭/그래스/사이코/미스트)·주요 아이템/특성·바디프레스/사이코쇼크/속임수/
무게 기반/자이로볼/고정 데미지/연속기. 테라스탈 없음(챔피언스 미존재).
미반영: HP 비례기 정밀치(만피 가정), 일부 마이너 특성.

## 데이터 출처
- 실시간 채용률: [championsbattledata.com](https://championsbattledata.com) 공개 API
- 정적 DB(한국어명·기술·종족값): @smogon/calc (MIT) 유래, 예시 APK에서 추출
- 스프라이트/아이콘: 예시 APK 자산 + PokeAPI sprites CDN 폴백 (개인 사용 목적)

## 파일 구성
```
champions-calc/
├── index.html      ← 실행 파일
├── app.js          ← 계산 엔진·인식·UI
├── matcher.js      ← 선출 화면 스프라이트 매칭 (카드 검출→전경 추출→마스크 MSE)
├── data.js         ← 정적 DB (포켓몬/기술/아이템/한국어명, 오프라인 폴백 채용률)
├── spriteindex.js  ← 스프라이트 매칭용 사전 계산 인덱스
├── serve.bat       ← (선택) 로컬 서버
└── assets/         ← 스프라이트·타입·아이템 아이콘
```
