파티 관리

이미지로 등록

게임의 내 파티 → 능력 화면과 스테이터스 화면을 캡처해서 순서 상관없이 넣어주세요. 능력 화면 → 특성·아이템·기술 / 스테이터스 화면 → 노력치·성격(화살표 색) 자동 등록.
여기에 Ctrl+V 붙여넣기 · 드래그&드롭 · 파일 선택
카드를 클릭하면 바로 세부 수정할 수 있어요. 파티는 브라우저에 자동 저장됩니다.

어떤 화면을 붙여넣나요?

① 상대 팀 인식 — 선출 화면

배틀 시작 전 선출 화면(싸울 포켓몬을 고르는 화면)을 캡처해서 계산기 탭 상단 바에 Ctrl+V로 붙여넣으세요. 오른쪽의 상대 포켓몬 6마리가 자동 인식되고, 실시간 채용률 세트가 적용됩니다.
캡처는 Alt+Print Screen(활성 창 캡처)이 편해요 — 캡처 직후 바로 Ctrl+V 하면 됩니다.

② 내 파티 등록 — 배틀팀 정보 화면

게임에서 트레이닝 → 팀 매니지먼트 → 본인 배틀팀을 공개한 뒤, 배틀팀 정보 화면을 열어주세요. 여기서 두 장을 캡처해 내 파티 탭의 등록 영역에 순서 상관없이 붙여넣으면 됩니다.
⚠ 배틀팀을 공개(등록)하지 않은 일반 파티 화면에서 캡처하면, 커스터마이징된 포켓몬 이름(닉네임)이 표시되어 포켓몬이 정상적으로 등록되지 않을 수 있어요. 배틀팀 정보 화면은 정식 이름으로 표시되므로 꼭 이 화면을 사용해주세요.
